# Настройки профиля

Настройки профиля относятся к аккаунту и не изменяют конфигурацию VPN-клиента.

## Что можно изменить

- Имя и доступные персональные данные.
- Пароль, если вход по паролю включён для аккаунта.
- Привязанные способы авторизации.
- Язык и другие доступные параметры интерфейса.

## Привязка аккаунта

Откройте «Кабинет → Профиль → Привязанные аккаунты». Привязывайте новый способ входа к текущему аккаунту, чтобы баланс и подписки остались в одном профиле.

## Удаление аккаунта

Создайте тикет из авторизованного кабинета. Сотрудник запросит подтверждение владельца и сообщит порядок обработки заявки.
