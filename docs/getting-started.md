# Начало работы

Для подключения понадобятся аккаунт XConnect, активная подписка, персональная ссылка и совместимое приложение.

## Перед началом

- Убедитесь, что выбранный тариф подходит вашей сети: обычный VPN для нормального интернета или LTE/Комбо для белых списков.
- Проверьте лимит устройств и срок подписки.
- Установите актуальную версию Incy с персональной страницы подписки или из официального магазина вашей платформы.

## Основные этапы

1. Войдите в существующий аккаунт или создайте новый.
2. Купите подписку либо выберите уже активную.
3. Скопируйте персональную HTTPS-ссылку.
4. Импортируйте ссылку в Incy.
5. Обновите подписку, выполните HTTP GET-проверку и подключитесь к рабочей локации.
