# Быстрый старт

## Как начать пользоваться VPN после оплаты?

1. Откройте в боте Кабинет → Подписки и выберите активную подписку.
2. Нажмите Подключиться / Показать ссылку.
3. Откройте выданную HTTPS-ссылку в браузере.
4. Установите предложенное приложение. На момент последнего тикета поддержка рекомендует Incy.
5. На странице подписки нажмите Добавить подписку либо вставьте ссылку в приложение вручную.
6. Обновите подписку в приложении, запустите проверку пинга и выберите рабочий сервер.

Ссылка персональная. Не публикуйте её и не передавайте посторонним.

## Где скачать приложение?

Список приложений и инструкции находятся на персональной странице подписки. Она открывается по ссылке из бота. Incy также доступен на incy.cc, в магазинах приложений и для Windows. По данным последнего тикета, собственное приложение XConnect выпущено для ПК, а Android-версия была в бете.

## Есть ли приложения для Windows, Linux, macOS, Android, iPhone и Android TV?

Да. На странице подписки есть варианты для Windows, Linux, macOS, Android, iOS и Android TV. Одна и та же ссылка на подписку используется на поддерживаемых платформах.

## Как добавить подписку вручную?

Скопируйте HTTPS-ссылку из бота, откройте Incy, нажмите + и выберите добавление подписки из буфера или по ссылке. Если кнопка на сайте не открывает приложение, вставьте ссылку вручную.

## Где найти ссылку повторно?

В боте: Подключиться → выбрать подписку → Показать ссылку. Не используйте ссылку другого пользователя и не пытайтесь угадать новый домен заменой старого — получите актуальную ссылку в боте.

## Можно ли пользоваться без Telegram?

После получения ссылки VPN работает независимо от Telegram. Кабинет доступен через веб-интерфейс, но первичная авторизация или привязка аккаунта может потребовать Telegram. Если Telegram временно не открывается, используйте уже добавленную подписку или заранее авторизованный веб-кабинет.
