# Установка Incy

Incy — основной клиент, который поддержка XConnect рекомендует для новых подключений. Старые клиенты могут не видеть часть серверов, не передавать HWID или не поддерживать используемые транспорты.

## Где скачать

Откройте персональную страницу подписки: она предложит подходящее приложение и инструкцию для вашей платформы. Incy также доступен через официальный сайт incy.cc и магазины приложений.

## Поддерживаемые платформы

- iPhone и iPad.
- Android-смартфоны и планшеты.
- Windows.
- macOS.
- Linux через совместимые клиенты.
- Android TV и другие поддерживаемые ТВ-устройства.

## После установки

Добавьте персональную ссылку, обновите подписку и выполните HTTP GET-проверку серверов. Если клиент сообщает, что приложение не поддерживается, создайте тикет: возможно, клиент не передаёт HWID.
