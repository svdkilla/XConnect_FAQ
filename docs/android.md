# Установка на Android

На Android используйте Incy из магазина приложений или с персональной страницы подписки.

## Инструкция

1. Установите Incy и запустите приложение.
2. Добавьте персональную ссылку через кнопку «+» или импорт из буфера обмена.
3. Разрешите создание VPN-подключения Android.
4. Обновите список серверов и включите HTTP GET-проверку.
5. Выберите подходящую локацию и подключитесь.

## Для стабильной работы

- Разрешите фоновую работу и отключите ограничение батареи для Incy.
- При зависании полностью завершите процесс и запустите клиент снова.
- Если AUTO не отвечает, выберите обычный сервер вручную.
- Для белых списков используйте только LTE/ROUTE-серверы своего тарифа.
