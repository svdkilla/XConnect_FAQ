# Установка на Linux

На Linux используйте клиент и формат конфигурации, предложенные на персональной странице подписки. Не все сторонние клиенты поддерживают AUTO, HWID и используемые транспорты.

## Общий порядок

1. Откройте персональную страницу подписки и выберите Linux.
2. Установите предложенный клиент для вашего дистрибутива.
3. Импортируйте HTTPS-подписку или совместимую конфигурацию.
4. Обновите список узлов и проверьте подключение к обычной локации.

## Если серверы импортируются не полностью

Создайте тикет и укажите дистрибутив, клиент и нужный формат. Для отдельных сценариев поддержка может подготовить raw VLESS, JSON или Mihomo-конфигурацию и проверить ограничение HWID.
