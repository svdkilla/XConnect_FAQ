# Установка на macOS

Установите Incy для macOS по ссылке с персональной страницы подписки или из официального источника клиента.

## Инструкция

1. Установите и откройте приложение.
2. Разрешите добавление VPN-конфигурации в системных настройках macOS.
3. Добавьте персональную ссылку XConnect.
4. Обновите подписку и проверьте серверы.
5. Выберите локацию и включите подключение.

## Если подключение блокируется системой

Проверьте раздел «Конфиденциальность и безопасность», удалите конфликтующие VPN-профили и перезапустите приложение. Если ошибка сохраняется, создайте тикет с версией macOS и скриншотом сообщения.
