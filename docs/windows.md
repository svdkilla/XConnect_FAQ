# Установка на Windows

Скачайте установщик Incy с персональной страницы подписки или официального источника клиента.

## Инструкция

1. Установите Incy и разрешите установку сетевого компонента, если Windows запросит подтверждение.
2. Скопируйте персональную ссылку и добавьте подписку через «+».
3. Обновите подписку, выполните HTTP GET-проверку и выберите сервер.
4. Включите VPN и проверьте открытие нужных сайтов.

## Если Incy зависает

Завершите процесс через Диспетчер задач и запустите его снова. Затем обновите подписку, отключите AUTO и попробуйте обычную локацию. При необходимости выполните сброс клиента и импортируйте ссылку заново.

## Что приложить к тикету

Укажите версию Windows и Incy, провайдера, тип сети, результат HTTP GET-пинга, выбранный сервер и точный текст ошибки. Не прикладывайте персональную ссылку целиком.
