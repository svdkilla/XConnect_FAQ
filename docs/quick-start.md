# Быстрый старт

После оплаты подписки выполните следующие действия. Обычно вся настройка занимает несколько минут.

## Подключение

1. Откройте «Кабинет → Подписки» и выберите активную подписку.
2. Нажмите «Подключиться» или «Показать ссылку».
3. Откройте персональную HTTPS-страницу подписки.
4. Установите предложенный клиент; основной рекомендуемый вариант — Incy.
5. Нажмите «Добавить подписку» либо вставьте ссылку в Incy вручную через кнопку «+».
6. Обновите подписку, включите HTTP GET-проверку, выберите рабочий сервер и подключитесь.

## Если автоматический сервер не работает

Обновите подписку и временно выберите обычную страну вручную. Для ограниченного мобильного интернета используйте только серверы LTE, ROUTE или «Локации под обход», если они входят в тариф.

> Не изменяйте домен внутри ссылки вручную и не используйте ссылку другого пользователя. Получайте актуальную ссылку только в своём кабинете.
